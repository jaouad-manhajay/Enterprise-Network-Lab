# Project Overview

## Description

This project aims to build a complete enterprise network infrastructure using Cisco vIOS routers, Cisco IOSv-L2 switches, FortiGate Firewall, pfSense Firewall, and Windows Server 2019.

The lab is designed to simulate a real company environment and demonstrate enterprise networking skills.

---

## Objectives

- Build a scalable enterprise network.
- Configure Layer 2 and Layer 3 networking.
- Implement VLANs.
- Configure Inter-VLAN Routing.
- Deploy OSPF.
- Configure FortiGate Firewall.
- Configure pfSense Firewall.
- Deploy Windows Server services.
- Configure DHCP, DNS and Active Directory.
- Monitor the infrastructure.
- Produce professional documentation.
